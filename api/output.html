Is the Intel Core i5-8250U @ 1.60GHz Good for Gaming?

This is a fairly old CPU that is no longer competitive with newer CPUs. But this CPU contains 4 Cores and 8 Threads. This CPU may not do well with games especially more recent releases. Best to check with your system requirements from the developer of the games you are interested to see if it is compatible. [More ...]
